Download and Set Up

The quick version for technical users

— One —

If you don't have these yet, install Node.JS and Git.

— Two —

Open your command line/terminal app in your preferred location (as Admin if the location requires write privileges) and type:
git clone https://github.com/Superalgos/Superalgos

— Three —

Change directories to Superalgos and type:
node run
(use node run minMemo for computers with 8GB of RAM or less)
(use node run noBrowser if you intend to access the backend from a remote computer)
(use node run noBrowser minMemo if both of the above are true)

— Four —

Follow the Welcome tutorial!
It's the ultimate onboarding experience, superior to all other resources available.
