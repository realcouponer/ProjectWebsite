Superalgos ProtocolIn conceptual terms, the protocol is a file with a specific format capable of holding all the information concerning any Strategy. The Superalgos Protocol allows for standardizing the way in which we describe Strategies, so that the tools we develop to create or modify, visualize, backtest and deploy Strategies can handle all Strategies in a standardized fashion. It also allows Strategies to be portable. In its current early version, the protocol is simply the description of a JSON object which defines the desired automation.
