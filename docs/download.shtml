Download and Set Up

The quick version for technical users

— One —

If you don't have it yet, install Node.JS.

— Two —

Open your command line/terminal app and type:
git clone https://github.com/Superalgos/Superalgos

— Three —

Change directories to Superalgos and type:
node run
(use node run minMemo for computers with less than 8GB of RAM)
